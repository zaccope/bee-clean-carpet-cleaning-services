# Bee Clean Carpet Cleaning Services

Lead-magnet preview site for prospect outreach.

## Stack
- Next.js 14 (App Router) + TypeScript + Tailwind CSS
- Resend contact form (set `RESEND_API_KEY` and `CLIENT_EMAIL`)
- GA4 analytics (set `NEXT_PUBLIC_GA4_ID`)

## Branding
- Honey yellow (`#F4C430`) + warm ink (`#1A1A1A`) on cream (`#FFFBEB`).
- Real logo lives in `public/images/LOGO.jpg`.

## Develop
```
npm install
npm run dev
```

## Deploy
Pushes to `main` auto-deploy via Vercel. Manual: `vercel --prod`.
